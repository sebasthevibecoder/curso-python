Curso de Python

Fundamentos prácticos para estudiantes de 2º de Bachillerato

Introducción a Python ▼

Python es un lenguaje de programación interpretado de alto nivel conocido por su simplicidad y legibilidad. Creado por Guido van Rossum en 1991, Python ha ganado enorme popularidad en campos como la ciencia de datos, inteligencia artificial, desarrollo web y automatización.

En este curso, exploraremos los fundamentos de Python y cómo aplicarlo en situaciones prácticas orientadas a estudiantes de bachillerato.

Práctica 1: Variables y Tipos de Datos ▼

Objetivos de Aprendizaje

Aprenderás a trabajar con variables, tipos de datos básicos y operaciones en Python.

Ejercicio Práctico

Crea un programa que calcule el área y perímetro de diferentes figuras geométricas.

Ejemplo de código:


# Calcular área de un círculo
radio = 5
pi = 3.14159
area = pi * radio**2
print(f"El área del círculo es: {area}")
Práctica 2: Estructuras de Control ▼

Objetivos de Aprendizaje

Aprenderás a utilizar condicionales (if, elif, else) y bucles (for, while) para controlar el flujo de tu programa.

Ejercicio Práctico

Crea un programa que determine si un número es primo y muestre todos los números primos hasta un límite definido.

Práctica 3: Funciones y Modularidad ▼

Objetivos de Aprendizaje

Aprenderás a crear funciones reutilizables y a organizar tu código de manera modular.

Ejercicio Práctico

Desarrolla una calculadora con funciones para las operaciones básicas y algunas avanzadas.

Práctica 4: Estructuras de Datos ▼

Objetivos de Aprendizaje

Aprenderás a trabajar con listas, diccionarios, tuplas y conjuntos para organizar datos.

Ejercicio Práctico

Crea un programa para gestionar una agenda de contactos con nombres, teléfonos y correos electrónicos.

Práctica 5: Proyecto Final ▼

Objetivos de Aprendizaje

Integrarás todo lo aprendido en un proyecto completo que resuelva un problema real.

Proyecto Final

Desarrolla una aplicación de consola que simule un sistema de gestión para una biblioteca, tienda o inventario.
